Moneris **Configuration -> IWL3**
- **##16** print out the configuration from a working terminal
- Swap terminal with Moneris using the Portal
- On new terminal input admin ID. Usually we put 20202 if the merchant does not know their admin ID
- **##17** register new terminal to the base with ETHERNET
- **##08** COMM TYPE:ETHERNET/TIP ENTRY:YES/TIP ENHANCED:YES/PERC1-2-3/RESTAURANT TYPE:1
- **##01** initialize the new terminal
- **##09** TABLE PROMPT:YES/TABLE NAME:YES
- **##18** enter the IP address and port numbers
- *some restaurants use ##83 and ##80 for restaurant options. Confirm with the print out if there are any values listed under these options.
- *If merchant using Touch Bistro. Add new serial number in touch bistro and remove the old one.
